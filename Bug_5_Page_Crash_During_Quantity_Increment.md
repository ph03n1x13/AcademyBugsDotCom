#### Title
Page crashes when user increases product quantity

#### Severity
High 

#### Description
The frontend freezes when a user tries to increase the product quality by selecting `Pink` or `Green` rom the `Select Color` option in product details

#### How to Reproduce
- Go to `https://academybugs.com/find-bugs/` 
- Select `Denim Coat` from the list  
- Select `Pink` or `Green` from the `Select Color` option 
- Click the `+` sign on the left side of the `Add to Cart` button 
- See the page freeze
#### PoC
[PoC Video](assets/videos/Denim_Page_Break.mp4)

#### Expected Result
The quantity should be increased as per the desired value without any frontend break
